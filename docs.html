Documentation

Unswipe is a zero-dependency carousel that delegates physics to CSS Scroll Snap and accessibility to Intersection Observer. Core bundle: < 2 KB minified.

Install

npm

npm install unswipe

Or copy dist/unswipe.js from the repository and load it as an ES module.

JavaScript

import { Unswipe } from 'unswipe';

const slider = new Unswipe(
  document.getElementById('carousel'),
  { align: 'start', label: 'Featured products', threshold: 0.5 },
);

slider.on('select', ({ index }) => {
  console.log('Active slide:', index);
});

Constructor

new Unswipe(root, options?, plugins?)
Parameter Type Description
root HTMLElement Carousel container. Receives scroll-snap styles and ARIA roles.
options SliderOptions Optional configuration. See Options.
plugins SliderPlugin[] Optional plugins initialized after the carousel is ready.

Returns a Slider instance.

Options

Option Type Default Description
slide string — CSS selector for slides relative to root. Defaults to direct children.
align 'start' | 'center' | 'end' 'start' Scroll-snap alignment anchor for each slide.
axis 'x' | 'y' 'x' Scroll axis. Horizontal uses scroll-snap-type: x mandatory.
label string — Accessible name applied as aria-label on the carousel root.
threshold number 0.5 Intersection ratio (0–1) required for a slide to be considered active.
behavior ScrollBehavior 'smooth' Default scroll behavior for next(), prev(), and scrollToIndex().

Instance API

Properties

Property Type Description
root HTMLElement The carousel container element.
slides readonly HTMLElement[] Current slide elements, re-collected on update().
index number Zero-based index of the active slide.

Methods

Method Description
next() Scroll to the next slide. No-op on the last slide.
prev() Scroll to the previous slide. No-op on the first slide.
scrollToIndex(index, behavior?) Scroll to a slide by index. Optional behavior overrides the default.
update() Re-scan the DOM for slides, re-attach observers, and emit 'update'.
destroy() Disconnect observers, destroy plugins, and clear event listeners.
on(event, handler) Subscribe to an event. Returns an unsubscribe function.

'select' { index: number; slide: HTMLElement } Active slide changes after scrolling or swiping.

JavaScript

import { Unswipe } from 'unswipe';
import { autoplay } from 'unswipe/plugins/autoplay';
import { navigation } from 'unswipe/plugins/navigation';
import { pagination } from 'unswipe/plugins/pagination';

const slider = new Unswipe(root, options, [
  navigation(),
  pagination(),
  autoplay({ delay: 5000 }),
]);

autoplay

Entry: unswipe/plugins/autoplay

Option Type Default Description
delay number 3000 Interval between slides in milliseconds.
pauseOnHover boolean true Pause while pointer is over the carousel or it has focus.
loop boolean true Return to the first slide after the last.

navigation

Entry: unswipe/plugins/navigation

Creates previous/next buttons and wires them to prev() and next().

Option Type Default Description
container HTMLElement auto-created Wrapper for the buttons. Inserted after the carousel root if omitted.
prevLabel string 'Previous slide' aria-label for the previous button.
nextLabel string 'Next slide' aria-label for the next button.
className string 'unswipe-nav' Base CSS class for buttons and container.

pagination

Entry: unswipe/plugins/pagination

Creates one dot per slide, synced via select and rebuilt on update.

Option Type Default Description
container HTMLElement auto-created Wrapper for the dot list.
className string 'unswipe-pagination' Base CSS class for the tablist and dots.
label string 'Carousel pagination' aria-label for the tablist.

Custom plugin

Implement the SliderPlugin interface:

import type { SliderPlugin } from 'unswipe';

export function logger(): SliderPlugin {
  let off: (() => void) | undefined;

  return {
    name: 'logger',
    init(slider) {
      off = slider.on('select', ({ index }) => {
        console.log('[unswipe] slide', index);
      });
    },
    destroy() {
      off?.();
    },
  };
}
Tip: Store the unsubscribe function returned by slider.on() in init and call it from destroy.

Dynamic mutations

Add or remove slides in the DOM, then call update(). The carousel re-collects slides, re-attaches the Intersection Observer, and emits 'update' for plugins like pagination.

Accessibility

Unswipe applies and maintains the following automatically:

  • role="carousel" on the root
  • aria-label on the root when label is set
  • aria-roledescription="slide" on each slide
  • aria-hidden="true" and tabindex="-1" on off-screen slides
  • Removal of aria-hidden and tabindex on the active slide
Best practice: Always pass a descriptive label option so screen-reader users understand the carousel's purpose.

React (client component)

import { useEffect, useRef } from 'react';
import { Unswipe } from 'unswipe';

export function Carousel({ children }) {
  const ref = useRef(null);

  useEffect(() => {
    const slider = new Unswipe(ref.current, { label: 'Products' });
    return () => slider.destroy();
  }, []);

  return <div ref={ref}>{children}</div>;
}
