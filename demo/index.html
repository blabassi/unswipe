Interactive guide

Unswipe

A zero-dependency carousel powered by CSS Scroll Snap and Intersection Observer. Every section below pairs live UI with the code that runs it.

< 2 KB core Zero dependencies CSS Scroll Snap IO-driven a11y

Install

npm

npm install unswipe

CDN / copy

import { Unswipe } from './assets/unswipe.js';

No runtime dependencies. Import the core (< 2 KB) and add plugins only when you need them.

  pagination(),

Accessibility

Applied automatically — try tabbing through slides above.

  • role="carousel" on the root
  • aria-label when label is set
  • aria-roledescription="slide" on each slide
  • aria-hidden + tabindex="-1" on off-screen slides

Frameworks & patterns

Unswipe is framework-agnostic — mount on any DOM node and call destroy() on teardown.

Web Component (live)

Element

class UnswipeCarousel extends HTMLElement {
  #slider;

  connectedCallback() {
    this.classList.add('carousel');
    this.#slider = new Unswipe(this, {
      label: this.getAttribute('label') ?? 'Carousel',
      align: 'start',
    });
  }

  disconnectedCallback() {
    this.#slider?.destroy();
  }
}

customElements.define('unswipe-carousel', UnswipeCarousel);

React (live)

Mounting…

React

import { useEffect, useRef } from 'react';
import { Unswipe } from 'unswipe';

export function Carousel({ slides, label = 'Carousel' }) {
  const ref = useRef(null);

  useEffect(() => {
    const root = ref.current;
    if (!root) return;

    const slider = new Unswipe(root, { label, align: 'start' });
    return () => slider.destroy();
  }, [label]);

  return (
    <div ref={ref} className="carousel">
      {slides.map((text) => (
        <div key={text} className="slide">{text}</div>
      ))}
    </div>
  );
}

<script setup>
import { onMounted, onUnmounted, ref } from 'vue';
import { Unswipe } from 'unswipe';

const root = ref(null);

onMounted(() => {
  slider = new Unswipe(root.value, { label: 'Products' });
});

onUnmounted(() => slider?.destroy());
</script>

  </div>

Custom plugin 
import type { SliderPlugin } from 'unswipe';

export function logger(): SliderPlugin {
  let off;
  return {
    name: 'logger',
    init(slider) {
      off = slider.on('select', ({ index }) => {
        console.log('[unswipe]', index);
      });
    },
    destroy() { off?.(); },
  };
}
