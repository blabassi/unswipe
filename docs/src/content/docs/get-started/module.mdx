## Install

```bash
npm install unswipe
```

Or import a built file / CDN copy:

```js
import { Unswipe } from './unswipe.js';
```

## Quick start

Slides default to **direct children** of the root element.

Optionally import the default stylesheet for usable chrome (gap, slides, nav, dots). Size slides with CSS tokens like `--unswipe-slide-min-width` (see [Slide sizes](../../guides/slide-sizes/)):

```js
import 'unswipe/style.css';
```

## Adding plugins

```ts
import { drag } from 'unswipe/plugins/drag';
import { navigation } from 'unswipe/plugins/navigation';

new Unswipe(root, { label: 'Gallery' }, [drag(), navigation()]);
```
