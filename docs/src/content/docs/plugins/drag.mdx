```ts
import { drag } from 'unswipe/plugins/drag';

new Unswipe(root, {}, [drag()]);
```

Mouse/pen drag-to-scroll. Touch stays native by default. Emits `pointerDown` / `pointerMove` / `pointerUp` with `{ x, y }`.

| Option | Default | Description |
| --- | --- | --- |
| `mouseOnly` | `true` | Ignore touch pointers |
| `threshold` | `10` | Pixels before a drag starts |

Pair with `{ dragFree: true }` for free scroll without snap.
