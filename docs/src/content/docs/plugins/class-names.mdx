---
title: Class names
description: Toggle selected, dragging, and in-view CSS classes.
---

```ts
import { classNames } from 'unswipe/plugins/classNames';

new Unswipe(root, {}, [classNames()]);
```

| Option | Default |
| --- | --- |
| `selected` | `is-selected` |
| `draggable` | `is-draggable` |
| `dragging` | `is-dragging` |
| `inView` | `is-in-view` |
