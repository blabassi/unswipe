---
title: Plugins
description: Tree-shakeable plugins that extend Unswipe.
---

Plugins implement `SliderPlugin` and ship as separate entry points so they never bloat the core. `init` may return a public API exposed via `slider.plugins()`.

```ts
import { Unswipe } from 'unswipe';
import { drag } from 'unswipe/plugins/drag';
import { loop } from 'unswipe/plugins/loop';
import { navigation } from 'unswipe/plugins/navigation';

new Unswipe(root, options, [loop(), drag(), navigation()]);
```
