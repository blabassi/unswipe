`slidesToScroll` controls how many slides `next()` / `prev()` and pagination groups advance. It does **not** set how many slides are visible — that is CSS (or the `slidesPerView` plugin).

```ts
new Unswipe(root, { slidesToScroll: 3 });
```

<Aside type="tip">
Show several slides but nudge one at a time with `slidesToScroll: 1` (the default).
</Aside>
