Plugins implement the `SliderPlugin` interface and ship as separate entry points so they never bloat the core. `init` may return a public API object exposed via `slider.plugins()`.

```ts
import { Unswipe } from 'unswipe';
import { autoplay } from 'unswipe/plugins/autoplay';
import { drag } from 'unswipe/plugins/drag';
import { loop } from 'unswipe/plugins/loop';
import { navigation } from 'unswipe/plugins/navigation';
import { pagination } from 'unswipe/plugins/pagination';

new Unswipe(root, options, [
  loop(),
  drag(),
  navigation(),
  pagination(),
  autoplay({ delay: 5000 }),
]);
```

## Drag

Mouse/pen drag-to-scroll. Touch stays native by default. Emits `pointerDown` / `pointerUp`.

| Option | Default | Description |
| --- | --- | --- |
| `mouseOnly` | `true` | Ignore touch pointers |
| `threshold` | `10` | Pixels before a drag starts |

Pair with `{ dragFree: true }` for free scroll without mandatory snap.

## Loop

Infinite carousel via edge clones + silent teleports. Keeps CSS Scroll Snap physics.

| Option | Default | Description |
| --- | --- | --- |
| `clones` | `1` | Clones per edge |

## Autoplay

| Option | Default | Description |
| --- | --- | --- |
| `delay` | `3000` | Interval in ms |
| `pauseOnHover` | `true` | Pause on hover/focus |
| `loop` | `true` | Jump to first when not using the loop plugin |

Uses `canScrollNext()` so it cooperates with the loop plugin.

## Navigation

Prev/next buttons. Disabled state follows `canScrollNext` / `canScrollPrev`.

| Option | Default | Description |
| --- | --- | --- |
| `prevLabel` | `"Previous slide"` | Accessible label |
| `nextLabel` | `"Next slide"` | Accessible label |
| `className` | `"unswipe-nav"` | CSS class prefix |
| `container` | sibling wrapper | Where buttons render |

## Pagination

Dots — one per snap group when `slidesToScroll > 1`.

| Option | Default | Description |
| --- | --- | --- |
| `label` | `"Carousel pagination"` | Tablist label |
| `className` | `"unswipe-pagination"` | CSS class prefix |
| `container` | sibling wrapper | Where dots render |

## Class names

| Option | Default |
| --- | --- |
| `selected` | `is-selected` |
| `draggable` | `is-draggable` |
| `dragging` | `is-dragging` |
| `inView` | `is-in-view` |

## Wheel

Maps vertical wheel to horizontal scroll.

| Option | Default | Description |
| --- | --- | --- |
| `speed` | `1` | Delta multiplier |

## Auto scroll

Continuous motion (distinct from interval autoplay). Exposes `{ play, stop }` via `plugins().autoScroll`.

| Option | Default | Description |
| --- | --- | --- |
| `speed` | `1` | Pixels per frame |
| `pauseOnHover` | `true` | Pause on hover |
| `reverse` | `false` | Reverse direction |

## Fade

Opacity crossfade. Best with full-width slides.

| Option | Default |
| --- | --- |
| `activeOpacity` | `1` |
| `inactiveOpacity` | `0.25` |

Import `unswipe/style.css` for default nav/dot styling (override with `--unswipe-*` tokens).

## Custom plugin

```ts
import type { SliderPlugin } from 'unswipe';

export function logger(): SliderPlugin {
  let off: (() => void) | undefined;
  return {
    name: 'logger',
    init(slider) {
      off = slider.on('select', ({ index }) => {
        console.log('[unswipe]', index);
      });
      return { ping: () => 'pong' };
    },
    destroy() {
      off?.();
    },
  };
}
```
