## Install

```bash
npm install unswipe
```

Or import a built file / CDN copy:

```js
import { Unswipe } from './unswipe.js';
```

## Quick start

Slides default to **direct children** of the root element.

<Playground
  id="ex-quick"
  label="Featured products"
  code={`import { Unswipe } from 'unswipe';

const slider = new Unswipe(document.getElementById('carousel'), {
  align: 'start',
  label: 'Featured products',
  threshold: 0.5,
});

slider.on('select', ({ index }) => {
  console.log('Active:', index);
});`}
/>
