## Install

```bash
npm install unswipe
```

Or import a built file / CDN copy:

```js
import { Unswipe } from './unswipe.js';
```

## Quick start

Slides default to **direct children** of the root element.

Optionally import the default stylesheet for usable chrome (gap, slides, nav, dots). Tokens are CSS custom properties — override `--unswipe-accent`, `--unswipe-gap`, and friends as needed:

```js
import 'unswipe/style.css';
```
