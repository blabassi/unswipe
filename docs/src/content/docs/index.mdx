## Why Unswipe?

<CardGrid>
  <Card title="Tiny core" icon="rocket">
    Core stays lean. Plugins are separate entry points so you only pay for what
    you use.
  </Card>
  <Card title="Native physics" icon="seti:css">
    CSS Scroll Snap handles kinetic scrolling on the compositor — no
    `requestAnimationFrame` loops.
  </Card>
  <Card title="Accessible by default" icon="approve-check">
    Scroll position drives `aria-hidden` and `tabindex` for the active slide.
  </Card>
  <Card title="Framework-agnostic" icon="puzzle">
    Mount on any DOM node. Works with Web Components, React, Vue, Svelte, or plain
    HTML.
  </Card>
</CardGrid>
